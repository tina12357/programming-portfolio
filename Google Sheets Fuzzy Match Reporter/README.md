Google Sheets Fuzzy Match Reporter
這是一個專為 Google Sheets 開發的 Apps Script 工具，用於自動偵測 A 欄中「疑似重複」的人名或資料。它具備模糊比對功能，能自動排除特定稱謂（如先生、小姐、律師等），並生成一份獨立的比對報告。

🌟 功能特點
模糊比對邏輯：不只是完全比對，支援「5 字重 4 字」、「3 字重 2 字」等相似度判定。

自動撇除稱謂：內建正則表達式，自動忽略「先生、小姐、女士、律師、教授、醫師」等後綴，提高比對準確度。

無損檢測：不會更動原始分頁的資料與格式。

獨立報告分頁：自動建立「重複檢查報告」工作表，列出：

疑似重複的群組 (Group)

原始資料所在的列號 (Row Number)

完整原始內容

排除稱謂後的比對字串

🚀 安裝與使用
1. 放入指令碼
開啟你的 Google 試算表。

點選選單 「延伸功能」 > 「Apps Script」。

將 generateFuzzyMatchReport.gs 的程式碼貼入編輯器中。

點選「儲存」並為專案命名。

2. 執行指令
在 Apps Script 編輯器上方，確認選取了 generateFuzzyMatchReport 函數。

點選 「執行」。

第一次執行時，Google 會要求授權存取你的試算表，請點選「查看權限」並同意。

3. 查看結果
回到試算表，腳本會自動建立一個名為 「重複檢查報告」 的新分頁。

根據報告中的「原始列號」，你可以快速回到主表處理重複資料。

⚙️ 比對邏輯說明
本工具預設的比對邏輯如下：

| 字串長度 (N) | 判定重複條件 (相同字數) | 範例 |
| :--- | :--- | :--- |
| **N >= 5** | 相同字數 >= 4 | 「國立台灣大學」 vs 「國立台北大學」 |
| **N = 3** | 相同字數 >= 2 | 「陳小明」 vs 「陳曉明」 |
| **N = 2** | 完全相同 | 「王明」 vs 「王明」 |
| **3 < N < 5** | 相似比例 >= 75% | 「王小明先生」 vs 「王小明」 |

註：比對前會先執行字串清理，移除所有空白及預設稱謂。

🛠 參數自定義
如果你需要增加更多要排除的稱謂，可以在程式碼中找到 titles 陣列進行修改：
```javascript
const titles = ["先生", "小姐", "女士", "律師", "教授", "長", "員", "醫師", "自定義稱謂"];
```
⚠️ 注意事項分頁名稱：腳本預設會讀取執行當下「活動中的分頁」。欄位限制：目前設計僅針對 A 欄 (從 A2 開始) 進行掃描。效能：由於模糊比對涉及 $O(n^2)$ 的交叉比對，若資料量超過 5,000 筆以上，執行時間可能會較長。
